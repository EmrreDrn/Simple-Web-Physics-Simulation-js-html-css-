# Simple-Web-Physics-Simulation-js-html-css-
This is a simple web physics simulation; whoever wants to can develop it further. HTML, JavaScript, and some CSS were used.
Your browser blocks loading local JS files (file://…) due to security rules called Same-Origin Policy. It doesn’t allow modules to be loaded directly from local files.

When you put your project on GitHub Pages (https://…), this problem won’t happen, because the files will be served over HTTP, not file://.

You can use Node.js or Python for the server; Node.js is recommended.
